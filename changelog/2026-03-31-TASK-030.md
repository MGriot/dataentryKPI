# Changelog - 2026-03-31

## [TASK-030] - Visual Formula Editor & Enhanced Exports

### Added
- **Visual Node Editor (Streamlit)**: Integrated a custom HTML/Canvas-based node graph editor for building KPI formulas visually. 
  - Replicates Tkinter logic with draggable nodes and smart sockets.
  - Bidirectional sync with formula strings.
  - Built-in KPI hierarchy search sidebar.
- **Individual Table Export**: Added the ability to export specific database tables (Plants, Hierarchy, Definitions, etc.) to CSV from both Tkinter and Streamlit GUIs.
- **Unified Export Backend**: Centralized export logic in `src/export_manager.py` for absolute consistency.

### Changed
- **Repository Reorganization**:
  - Moved project management files (`tasks.json`, `stage.md`) to `conductor/`.
  - Moved user configuration files (`settings.json`, `user_constants.json`) to `user_config/`.
  - Updated path resolution in `src/config/settings.py` and Tkinter `settings_tab.py`.
- **Streamlit Data Center**: Redesigned the UI to prioritize individual table exports at the top of the page.

### Fixed
- **Streamlit Component Loading**: Applied strict v1 handshake protocol (`isStreamlitMessage`, `apiVersion: 1`) to resolve "trouble loading component" errors.
- **Iframe Race Condition**: Ensured `setFrameHeight` triggers only after successful render event.
