## Task ID & Title
TASK-010: Smart Data Entry: Auto-population & Visual Feedback

## Timestamp
[13:45:00]

## The "Why"
To provide immediate feedback to users and reduce manual effort, the Data Entry page now automatically recalculates dependent KPIs as values are changed. Visual feedback through color-coding (Light Blue for Calculated, White for Manual, Light Green for Derived) helps users distinguish the source of each target value at a glance, improving data entry accuracy and efficiency.

## Verification
- **Styles**: Added `Formula.TLabelframe`, `Manual.TLabelframe`, and `Derived.TLabelframe` to the application's global styles.
- **Interactivity**: Implemented `tk.StringVar` tracing on target entry fields to trigger real-time recalculations.
- **Node Engine Integration**: Connected the UI recalculation loop to the `KpiDAG` evaluation engine, allowing visual formulas to update live on the page.
- **Visual State**: Updated `_update_sub_kpi_target_field_state` to dynamically apply styles based on the KPI's calculation mode and hierarchy role.
