# Changelog: 2026-03-11-TASK-013

## Task ID & Title
TASK-013: Node Editor: Role-Based Sockets & Dynamic Connections

## Timestamp
[15:15:00]

## The "Why"
Initial node editor connections were unreliable and lacked semantic clarity for non-commutative operations like division. Refactoring the connection logic to use proximity detection and labeled sockets (e.g., Numerator vs. Denominator) ensures that users can precisely define calculation logic and that connections always snap to the intended inputs.

## Verification
- **Sockets**: Implemented role-based labels for input sockets.
- **Connections**: Refined proximity detection for reliable snapping during drag-and-drop.
- **Deletion**: Added full support for deleting nodes and edges using the `Delete` key or toolbar button.
