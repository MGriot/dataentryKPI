TASK-028: Global Split Management & Preset Generation

## Timestamp

## The "Why"
Manual editing of JSON repartition values was prone to errors and slow. Implementing automatic preset generation for Months, Quarters, and Weeks provides a baseline that users can then customize, significantly improving the user experience in management of standardized distributions.

## Verification
- **UI**: Added "✂️ Global Splits" tab to `KpiManagementTab` with detailed JSON preview pane.
- **Presets**: Implemented `_on_preset_change` in `SplitEditorDialog` to generate default values based on selected Logic and Profile.
- **Date Logic**: Added logic to calculate the exact number of ISO weeks (52/53) based on the target year for accurate week-based splits.
