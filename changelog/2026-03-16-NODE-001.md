# [NODE-001] · Enhanced Node Editor

**Timestamp**: 2026-03-16 23:45 Local Time  
**Author**: Ralph  

## Why
The Node Editor needed visual and functional improvements to match the complexity of real-world KPI formulas and improve user experience.

## What Changed
- `src/gui/node_editor.py`:
    - **Hierarchical KPI Selection**: Replaced the flat listbox with a recursive `ttk.Treeview` that matches the system's hierarchy.
    - **Orthogonal Connections**: Changed edge drawing logic to use 90-degree orthogonal lines instead of diagonals.
    - **Dynamic Operator Inputs**: Operators like Sum and Multiplication now support a configurable number of input sockets.
    - **Edit Node Feature**: Added a toolbar button and double-click action to edit constants or operator configurations.
    - **Adaptive UI**: Nodes now dynamically adjust their height based on the number of input sockets.
